식품 R&D AI 프롬프트 치트시트

식품 개발 실무에서 바로 복사해서 쓸 수 있는 AI 프롬프트 템플릿 모음입니다.
상황별로 검증된 프롬프트를 활용하세요.

📌 6개 카테고리📝 18개 템플릿🎯 난이도별 분류

🏆 프롬프트 황금 규칙 5가지

1
출처를 요구하라 — "식품공전/첨가물공전 원문 근거로 답변해줘"
2
불확실성을 표기하라 — "확인 불가능한 정보는 '미확인'으로 표시해줘"
3
역할을 부여하라 — "너는 식품안전 전문가/10년차 R&D 연구원이야"
4
출력 형식을 지정하라 — "표 형식으로 / JSON으로 / 3단계로 정리해줘"
5
범위를 한정하라 — "국내 식품공전 기준으로 / 2024년 이후 데이터로"

🔬 원료 검증 및 규격 확인

원료의 사용 가능 여부, 규격, 기준을 확인할 때 사용합니다.

원료 사용 가능 여부 확인기초
"[원료명]이 한국 식품공전에서 [식품유형]에 사용 가능한지 확인해줘.
반드시 식품첨가물공전 원문을 근거로 답변하고, 확인 불가능한 정보는 '미확인'으로 표기해줘."
💡 TIP [원료명]과 [식품유형]을 구체적으로 — 예: 스테비올배당체, 탄산음료
원료 사용 기준 비교표중급
"다음 감미료의 식품공전 사용 기준을 표 형식으로 비교 정리해줘: [원료1, 원료2, 원료3].
비교 항목: 허용 식품유형, 사용 한도(g/kg), 병용 제한 사항.
식품첨가물공전 별표3을 근거로 하되, 원문 미확인 시 '확인 필요'로 표기해줘."
💡 TIP 표 형식을 요청하면 비교 분석에 매우 효과적
수입 원료 인허가 확인심화
"너는 식품안전 규제 전문가야. [수입 원료명]의 한국 수입 시 다음을 확인해줘:
1) 식품공전 등재 여부 2) 잔류 기준 3) 필요 서류 (건강기능식품법/수입식품안전관리특별법 기준)
불확실한 항목은 '확인 필요 — [담당기관]에 문의 권장'으로 표기해줘."
💡 TIP 규제 관련은 반드시 담당기관 문의 경로까지 요청

⚗️ 배합 설계 및 처방 개발

음료, HMR 등 제품 배합 설계 시 AI를 활용하는 프롬프트입니다.

음료 배합 초안 설계기초
"[제품 유형] 배합 초안을 설계해줘.
조건: Brix [범위], pH [범위], 주요 원료 [원료 리스트]
결과는 원료명 / 배합비(%) / 역할 표 형식으로 정리해줘.
식품공전상 사용 제한이 있는 원료는 별도 표시해줘."
💡 TIP Brix, pH 등 물리화학적 조건을 명시하면 실현 가능한 배합을 얻을 수 있음
기존 제품 개선 배합중급
"아래 기존 배합을 분석하고 개선안을 제안해줘:
[기존 배합 데이터 붙여넣기]

개선 목표: [당류 30% 저감 / 원가 절감 / 유통기한 연장]
제약 조건: [알레르기 원료 제외 / 비건 대응]
변경 원료의 식품공전 적합성을 반드시 확인하고 답변해줘."
💡 TIP 기존 배합 데이터를 직접 붙여넣으면 맥락을 정확히 이해함
물성 예측 기반 배합 최적화심화
"너는 음료 R&D 15년차 배합 전문가야.
아래 배합에서 [목표 물성: 점도 200cP, pH 3.5±0.2]을 달성하기 위한 최적화안을 제시해줘.
[배합 데이터]

각 원료의 pH 완충능(β), pKa 값을 고려하고,
수분 밸런스(Water Balance)가 100%가 되도록 검증해줘.
계산 과정을 단계별로 보여줘."
💡 TIP 계산 과정 요청 시 AI의 논리 오류를 직접 검증 가능

📊 시장 분석 및 트렌드

시장 동향, 소비자 인사이트, 경쟁 제품 분석에 활용합니다.

카테고리 트렌드 분석기초
"[음료/HMR/과자] 카테고리의 최근 트렌드를 분석해줘.
포함 항목: 1) 소비 키워드 Top 5 2) 성장 카테고리 3) 주요 타겟층 변화
출처와 데이터 시점을 반드시 명시하고, 불확실한 수치는 '추정'으로 표기해줘."
💡 TIP AI의 시장 데이터는 학습 시점 한계가 있으므로 반드시 출처 확인
경쟁 제품 비교 분석중급
"다음 제품들을 비교 분석해줘: [제품A, 제품B, 제품C]
비교 항목: 포지셔닝, 주요 원료, 가격대, 타겟, 차별화 포인트
표 형식으로 정리하고, 각 항목의 정보 출처를 함께 기재해줘.
온라인에서 확인 불가한 정보는 '현장 확인 필요'로 표기해줘."
💡 TIP 경쟁사 정보는 특히 할루시네이션이 잦으므로 출처 요구 필수
소비자 페르소나 기반 NPD 기회 도출심화
"너는 식품마케팅 10년차 브랜드 매니저야.
아래 소비자 데이터를 기반으로 [카테고리] 신제품 기회를 도출해줘:
[소비자 조사 데이터 / 트렌드 키워드]

결과 형식: 1) 페르소나 프로필 2) 미충족 니즈 3) 컨셉 제안 3개 4) 각 컨셉의 시장 리스크
데이터에 근거하지 않은 추측은 '가설'로 명확히 구분해줘."
💡 TIP 소비자 데이터를 직접 제공하면 근거 있는 인사이트를 얻을 수 있음

📜 표시·광고·규제 검증

식품표시, 광고 문구, 법규 적합성을 확인할 때 사용합니다.

식품 표시사항 적합성 검토기초
"[제품 유형]의 식품등의 표시기준에 따른 필수 표시사항을 정리해줘.
식품등의 표시·광고에 관한 법률과 시행규칙을 근거로 답변하고,
각 항목의 법적 근거 조항을 함께 기재해줘."
💡 TIP 법률명과 조항 번호를 요청하면 교차 검증이 훨씬 쉬워짐
광고 문구 법적 적합성중급
"다음 광고 문구가 식품표시광고법 위반 소지가 있는지 검토해줘:
\"[검토할 광고 문구]\"

검토 기준: 1) 질병 예방/치료 암시 여부 2) 기능성 표시 적법성 3) 비교 광고 적법성
위반 가능성이 있는 표현과 수정 대안을 함께 제시해줘.
판단이 어려운 부분은 '식약처 사전 심의 권장'으로 표기해줘."
💡 TIP 광고 심의는 AI 판단만으로 부족 — 반드시 전문가 확인 필요
영양성분 표시 계산중급
"아래 배합 데이터를 기반으로 영양성분 표시값을 계산해줘:
[배합 데이터 + 각 원료의 영양성분 정보]

1회 제공량 [X]ml 기준으로 열량, 탄수화물, 당류, 단백질, 지방, 나트륨을 계산하고,
식품등의 표시기준에 따른 표시값 허용오차 범위도 함께 안내해줘.
계산 과정을 단계별로 보여줘."
💡 TIP 계산 과정을 요청해야 오류를 직접 검증 가능

🏭 HACCP 및 품질 관리

HACCP 문서, 공정 설계, 위해 요소 분석에 활용합니다.

위해 요소 분석 (HA)중급
"[제품 유형]의 제조 공정에서 발생 가능한 위해 요소를 분석해줘.
공정 단계: [원료 입고 → 계량 → 혼합 → 살균 → 충전 → 포장]

각 단계별 생물학적/화학적/물리적 위해 요소를 표로 정리하고,
CCP(중요관리점) 판단 근거도 함께 제시해줘.
CODEX 가이드라인 기준으로 답변해줘."
💡 TIP 공정 단계를 구체적으로 나열할수록 정확한 분석 가능
유통기한 설정 실험 설계심화
"[제품 유형, 보존 조건]의 유통기한 설정을 위한 가속 실험 설계안을 제시해줘.
포함 항목: 1) 보관 온도 조건 2) 측정 항목 및 주기 3) 판정 기준
4) 가속 계수(Q10) 적용 시 계산 과정

식약처 유통기한 설정 가이드라인을 근거로 하되,
가이드라인에 명시되지 않은 부분은 '자체 판단 필요'로 표기해줘."
💡 TIP 유통기한 설정은 법적 책임이 따르므로 AI 결과를 시작점으로만 활용

📝 보고서 및 발표 자료

R&D 보고서, 경영진 보고, 발표 자료 작성에 활용합니다.

NPD 보고서 초안 작성기초
"아래 정보를 바탕으로 신제품 개발 보고서 초안을 작성해줘:
제품 컨셉: [컨셉] / 타겟: [타겟] / 주요 원료: [원료]

보고서 구성: 1) 개발 배경 2) 시장 현황 3) 제품 컨셉 4) 원료 구성 5) 예상 원가 6) 출시 일정
각 섹션당 300자 이내로 작성하고, 데이터 미확인 부분은 [TBD]로 표시해줘."
💡 TIP 분량 제한과 불확실 표기 규칙을 미리 설정하면 실무에 바로 활용 가능
경영진 보고용 1-pager중급
"아래 NPD 프로젝트 내용을 경영진 보고용 1-pager로 요약해줘:
[프로젝트 상세 내용]

구성: 핵심 메시지(1문장) → 시장 기회 → 제품 차별점 → 예상 매출 → 핵심 리스크 → 다음 단계
총 500자 이내, 숫자와 데이터 중심으로 간결하게 작성해줘."
💡 TIP "경영진 보고용"이라는 맥락을 주면 AI가 핵심 중심으로 압축함
