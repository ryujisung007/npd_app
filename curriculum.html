AI와 식품 R&D: 기초 이해

  • ChatGPT, Gemini, Claude의 특성을 비교한다
1차시: 생성형 AI란 무엇인가90분
LLM의 기본 구조, 토큰화, 확률 기반 텍스트 생성 원리를 이해합니다. ChatGPT/Gemini/Claude의 특성을 비교하고 식품 R&D 적용 가능 영역을 탐색합니다.
이론토론: AI가 식품 산업을 바꿀까?
2차시: AI의 함정 — 할루시네이션90분
AI가 만들어내는 정보 왜곡(Hallucination)의 원리와 식품 분야 실제 사례를 학습합니다. 프롬프트 평가기를 활용한 실습을 진행합니다.
이론실습: 프롬프트 평가기퀴즈
📄 할루시네이션 교안 (HTML)🎯 프롬프트 평가기 (웹앱)
M2
프롬프트 엔지니어링 실전
📅 2주차⏱️ 180분 (2차시)🎯 기초~중급
▼
학습 목표
  • 프롬프트 황금 규칙 5가지를 실무에 적용한다
  • 페르소나 프롬프트를 설계하고 활용한다
  • 식품 R&D 상황별 프롬프트 템플릿을 작성한다
3차시: 프롬프트 설계 원칙90분
프롬프트 황금 규칙(역할 부여, 출처 요구, 불확실성 표기, 구체성, 출력 형식)을 학습하고, Zero-shot / Few-shot / Chain-of-Thought 기법을 실습합니다.
이론실습: 치트시트 활용
📋 프롬프트 치트시트 (HTML)
4차시: 페르소나 & 역할 기반 프롬프트90분
식품마케터, R&D 연구원, 품질관리 담당자 등 다양한 페르소나를 설계하고 프롬프트에 적용합니다. 같은 질문에 페르소나별 답변 차이를 비교 분석합니다.
실습: 페르소나 설계토론: 결과 비교
📝 페르소나 스크립트 작성 과제
M3
식품 데이터와 AI: 공전·법규 활용
📅 3주차⏱️ 180분 (2차시)🎯 중급
▼
학습 목표
  • 식품안전나라 API를 활용하여 데이터를 수집한다
  • 식품공전·첨가물공전 데이터를 AI 프롬프트에 활용한다
  • AI 답변을 공인 데이터베이스로 교차 검증한다
5차시: 식품안전나라 API 실습90분
식품안전나라 I1250 API를 활용하여 식품 인허가 데이터를 조회합니다. API 키 발급, 파라미터 설정, 결과 해석을 실습합니다.
실습: API 호출이론: REST API 기초
6차시: 공전 데이터 기반 프롬프트90분
수집한 공전 데이터를 AI 프롬프트에 직접 제공하여 정확도를 높이는 방법을 학습합니다. "데이터 주입 → AI 분석 → 결과 검증" 워크플로를 실습합니다.
실습: 데이터 기반 프롬프트퀴즈
M4
RAG: 검색 증강 생성 활용
📅 4주차⏱️ 180분 (2차시)🎯 중급~심화
▼
학습 목표
  • RAG(Retrieval-Augmented Generation)의 원리를 이해한다
  • 식품 문서(공전, 규격서)를 벡터 DB에 저장하고 검색한다
  • RAG 기반 질의응답 시스템을 구축한다
7차시: RAG 개념과 아키텍처90분
RAG의 작동 원리(문서 임베딩 → 벡터 DB 저장 → 검색 → 생성)를 학습합니다. 할루시네이션 방지에서 RAG의 역할을 이해합니다.
이론토론: RAG vs Fine-tuning
8차시: 식품 RAG 시스템 실습90분
ChromaDB + Gemini/OpenAI를 활용하여 식품공전 문서 기반 RAG 시스템을 구축합니다. Streamlit UI를 통한 질의응답 시스템을 만듭니다.
실습: ChromaDB + Streamlit
M5
AI 기반 식품 개발 워크플로
📅 5주차⏱️ 180분 (2차시)🎯 심화
▼
학습 목표
  • AI를 NPD(신제품개발) 전 과정에 통합 활용한다
  • 시장 분석 → 컨셉 개발 → 배합 설계 → 규격 검증 파이프라인을 구축한다
  • Streamlit 기반 AI 도구를 배포한다
9차시: AI 기반 NPD 파이프라인90분
시장 트렌드 분석(Naver Shopping API) → AI 컨셉 제안 → 배합 초안 생성 → 규격 자동 검증까지 이어지는 전체 워크플로를 설계합니다.
이론실습: 파이프라인 설계
10차시: Streamlit 대시보드 구축90분
음료 R&D 플랫폼을 Streamlit으로 구축합니다. DALL-E 이미지 생성, Gemini 배합 에이전트, 마케팅 컨셉 플로를 통합합니다.
실습: Streamlit 앱 개발
M6
최종 프로젝트 & 발표
📅 6주차⏱️ 180분 (2차시)🎯 종합
▼
학습 목표

  • 프로젝트 결과를 경영진 보고 형식으로 발표한다
  • AI 활용의 한계와 윤리적 고려사항을 정리한다
11차시: 프로젝트 작업 & 멘토링90분
팀별 최종 프로젝트(AI 활용 신제품 개발 제안서) 작업을 진행합니다. 강사 멘토링과 팀 간 피드백을 교환합니다.
